黑加手环自定义布局教程

发表于 2019-06-26 | 阅读次数:

在最开始提一句：不支持iOS
准备工作：
一台手机
一台电脑
MT文件管理器2.0（酷安下载地址）
黑加的apk(酷安下载地址）
一颗勇于创新的心
开始创作：
1.打开黑加的apk
2.进入assets文件夹
3.你会看到很多文件

5.现在选择你所要修改的布局文件并打开它
6.没错，你现在看到的是一堆字，为了方便起见你可以对他进行排版
（因为太难了，我临时改了教程）
6.打开电脑，把你要改动的布局传输到电脑上
7.先介绍如何改变元素位置/颜色/大小，请继续往下看，想要添加元素的后面会讲
8.让我们打开一个软件（感谢俄罗斯团队的开发）
（我之前改布局是纯手工改的，昨天才发现了这个神器）
（下载地址）
9.解压下载的文件，运行HeyPlusSurface.exe
10.注意：（本软件原先只为俄罗斯版本设计，因此部分功能在官方固件上不显示或不起作用，如果你添加了一个元素导致了所有元素的请不要慌张，删除不该添加的元素即可恢复手环正常）

| ， | 反正就是用不了 |
| temp_realtime | 俄罗斯中天气显示组件，官方固件启用会导致所有元素消失 |
| temp_forecast | 同上 |
| weather_city | 同上 |
| weather_type_rt | 同上 |

11.正式开干
在软件中点击上栏上的 Open，然后单击 Custom Surface… ，选择你要修改的元素
如果你选择要修改的是官方表盘，那么软件会加载官方表盘背景，如果你选择了自定软件会加载一张米环4中的表盘背景（你可以通过修改软件目录下Board文件夹内board0.bmp来改变成自己的背景）
13.基础操作介绍
软件左侧是你的表盘预览区，预览区的右侧是你的布局中已添加元素的总览，用鼠标选择一个元素，你可在下方的区域内设置此元素，其中包括position（位置）font选项中的size（大小，注意Font72在官方固件中不可用）和color（颜色），同时你可以在右侧的选择框内选择你要的元素，点击上栏中的add将元素加入你的布局，或选择已有的元素单击delete来删除。
14.完事以后的做法
单击上栏中的save，选择布局保存位置，保存布局，传输到手机，用mt文件管理器将其替换apk中的布局，然后签名（在apk上单击，选择功能，选择签名），然后进行安装（安装失败的请将原来的黑加app删除），最后打开安装好的黑加app，选择你修改的布局，进行安装（请选择自己修改的表盘布局所对应的原来的表盘预览图进行安装），如果你原来已经安装了该布局，请删除这个表盘重新安装。
如果不想弄/懒得弄/没有电脑，欢迎来找Shc，有空的话会帮忙的
